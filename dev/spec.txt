# The FRAME spec

## Visual
- Resolution: 64x64
- Colors: 8

## Architecture
- Memory: 16-bit (0x0000-0xFFFF)
- Registers: 4

## Memory Mapping
- 0x0000-0xEFFF: program memory

## Instruction Set
### hlt
Halts execution

### mov [r], [v]
Moves value [v] into [r]

### jmp [a]
Unconditional jump to address [a]

### jmpc [a]
Jumps to address [a] if the conditional flag is non-zero

### equ [a], [b]
Compares two values for equality and sets the conditional flag accordingly

### not <r>, <v>
Logical negation (zero becomes one, non-zero becomes zero)
- If no operands are given, negates the conditional flag
- Otherwise, negates [v] and stores the result into [r]

### add [r], [a], [b]
Adds values [a] and [b] and stores the result into [r]
[a] and [b] are treated as signed

## Assembler
- Instructions: mov, add, sub, etc.
- Labels: @label
- Numbers: 1, -6, 0xf3, 0b0110, 0o777, etc.
- Identifiers: my_Cool_Identifier_3, r0, etc.
